# Verification

## Local Commands

```bash
npm install
npm run dev
npm run build
```

## Expected Result

- `npm run dev` starts Vite and shows the stats panel page.
- `npm run build` finishes without TypeScript or bundling errors.

## Manual Checklist

- [ ] Layout structure matches the target panel hierarchy.
- [ ] Card spacing and typography are visually consistent.
- [ ] Responsive behavior stacks cards on small screens.
- [ ] No runtime errors in browser console.
